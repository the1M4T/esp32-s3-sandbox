# esp32-s3-sandbox

Hardware and software experiments on the ESP32-S3 microcontroller. Focusing on low-level configuration, signal processing, and sensor integration.

## 🎯 Overview
This repository serves as a technical proving ground for bridging raw hardware capabilities with custom software. The primary objective is to build a robust foundation in embedded systems, testing everything from basic GPIO and communication protocols to early-stage robotics mechanics and machine learning algorithms built from scratch. 

## ⚙️ Hardware Setup
* **Target MCU:** ESP32-S3 (ESP32-S3-WROOM-1-N16R8)
* **Core Architecture:** Xtensa® dual-core 32-bit LX7
* **Key Features Explored:** Vector instructions for ML, native USB, signal intelligence
